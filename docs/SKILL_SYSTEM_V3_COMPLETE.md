## 一、设计总览

技能体系 v3 以**卡牌品质**决定特技槽位，每位英雄拥有 **1 个普通技能** + **0~1 个品质特技**：

- **普通技**：开局即拥有，**不可升级**，提供基础战术能力
- **特技**：按品质解锁（稀有/史诗/传奇），**可升至 5 级**，效果按 `base × (1 + 8%×(L-1))` 成长
- **取消近战/远程战斗逻辑**：`attackRange` 仅影响表现；`attackSpeed` = 弹道飞行速度；`fireRate` = 发射频率

当前共 **67** 个技能，覆盖 **37** 位英雄。

## 四、弹道与防御规则

### 4.1 两种弹道

| 弹道 | 代号 | 典型来源 | 视觉特征 |
|:---|:---:|:---|:---|
| **平直** | `flat` | 普通攻击技、双联/连锁特技 | 直线射向目标，无抛物线 |
| **高抛** | `arc` | 传奇攻击特技「高空重击」、亡灵收割 | 强抛物线，高空砸击落地 |

### 4.2 防御分层

| 防御类型 | 可挡弹道 | 说明 |
|:---|:---:|:---|
| **普通铁壁**（普通防御技） | 仅 `flat` | 无法抵挡特技高抛；高抛攻击穿透 |
| **稀有单格护墙**附加减伤 | 仅 `flat` | 护墙挡位 + 平直减伤 |
| **史诗三格盾带** | `flat` + `arc` | 范围内友军可挡高抛 |
| **传奇全局圣域** | `flat` + `arc` | 全队减伤，可挡高抛 |

### 4.3 对抗示例

- 重步兵（铁壁 + 单格护墙）：对平直攻击约 **16%** 减伤，对高抛攻击 **0%** 减伤
- 龙焰女王（高空重击）：攻击弹道为 `arc`，可穿透普通铁壁
- 圣盾领主（铁壁 + 全局圣域）：自身对平直最高约 **40%** 减伤，对高抛约 **33%** 减伤

### 4.4 配表字段

| 字段 | 位置 | 含义 |
|:---|:---|:---|
| `effects[].blocksTrajectory` | 防御效果 | `["flat"]` 或 `["flat","arc"]` |
| `attackTrajectory` | 攻击特技 | `"arc"` 表示高抛攻击 |
| `bypassesNormalDefense` | 攻击特技 | `true` = 穿透普通铁壁 |
| `combatRules.trajectory` | skill.json 全局 | 弹道规则说明 |

## 五、英雄属性公式

```
attack(L)      = round(attackL1 × 1.15^(L-1))
unitHp(L)      = round(unitHpL1 × 1.15^(L-1))
buildingHp(L)  = round(buildingHpL1 × 1.15^(L-1))
fireRate(L)    = round(fireRateL1 × 1.02^(L-1), 2)    // 发射频率
attackSpeed(L) = round(attackSpeedL1 × 1.03^(L-1), 2) // 弹道飞行速度
attackInterval = 2.2 / fireRate                         // 攻击间隔（秒）
```

## 六、特技升级消耗

效果曲线：`effect(L) = base × (1 + 0.08 × (L-1))`

| 升级 | 基础碎片 | 普通 | 稀有 | 史诗 | 传奇 |
|:---:|:---:|:---:|:---:|:---:|:---:|
| L1→L2 | 5 | 5 | 8 | 12 | 20 |
| L2→L3 | 8 | 8 | 12 | 20 | 32 |
| L3→L4 | 12 | 12 | 18 | 30 | 48 |
| L4→L5 | 18 | 18 | 27 | 45 | 72 |

金币消耗 ≈ 同级英雄升级金币 × **15%**（见 `heroLevel.json`）。

## 七、效果类型说明

| 效果类型 | 适用 | 说明 |
|:---|:---|:---|
| `atkPct` | 攻击 | 攻击力百分比加成 |
| `damageReductionPct` | 防御 | 自身减伤；配合 `blocksTrajectory` |
| `allyDamageReductionPct` | 防御 | 友军范围减伤（史诗三格盾带） |
| `globalDamageReductionPct` | 防御 | 全队减伤（传奇全局圣域） |
| `splashPct` | 攻击 | 邻格溅射伤害比例 |
| `extraTargets` | 攻击 | 额外命中目标数 |
| `chainTargets` | 攻击 | 连锁命中格数 |
| `projectileArc` | 攻击 | 标记高抛弹道（触发 arc 攻击） |
| `executeThreshold` | 攻击 | 低血斩杀阈值 |
| `wallTiles` | 防御 | 护墙覆盖格数 |
| `healPct` / `healAllyPct` / `healAreaPct` | 补给 | 回血比例 |
| `healFull` | 补给 | 满血恢复 |
| `fireRatePct` / `projectileSpeedPct` | 加速 | 攻速/弹速提升 |
| `teamFireRatePct` | 加速 | 全队攻速 |
| `revealAdjacent` | 加速 | 翻开相邻格 |
| `deployGold` | 补给 | 翻开获得金币（采矿机） |

## 十、战斗接入说明

| 模块 | 职责 |
|:---|:---|
| `skill.json` | 技能定义、效果、弹道规则、`levelCurve` |
| `heroBattle.json` | 英雄 `combatStats`、技能槽映射 |
| `bond.json` | 阵营与定位羁绊 |
| `battle-skill-runtime.js` | 汇总 `trajectoryDefense`、`primaryAttackTrajectory`、战斗修饰符 |
| `battle-rules.js` | 按 `attackTrajectory` 结算减伤与伤害 |
| `skill-config.js` | 配表加载与查询 |

生成命令：`python3 scripts/gen-skill-bond-config.py`
